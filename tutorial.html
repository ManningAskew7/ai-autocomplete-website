‚ú® Core Features

Text Autocomplete

Ctrl + Space Trigger Autocomplete
1. Place your cursor where you want to write
2. Press Ctrl+Space to trigger AI suggestions
3. Wait for the AI to generate 5 unique completions
4. Press Tab to cycle through suggestions
5. Press ‚Üí to accept or Esc to dismiss

Completion Modes

Shift + Alt + M Switch Completion Mode
Short Mode: 5-20 words for quick completions
Medium Mode: 20-40 words for standard paragraphs
Long Mode: 50-100 words for detailed content

Press the keybind anytime to cycle through modes. Your current mode appears briefly when changed.

AI Text Rewriting

Alt + Shift + R Rewrite Selected Text
1. Select any text you want to improve
2. Press Alt+Shift+R
3. AI generates 5 different rewrites
4. Click a number or use arrow keys to preview
5. Press Enter to apply or Esc to cancel

üîß Troubleshooting

Extension not working?

  • Check if you've accepted the privacy consent
  • Verify your API key has credits
  • Try switching injection modes
  • Use manual injection: Ctrl+Shift+Space

No suggestions appearing?

  • Ensure you have an active internet connection
  • Check OpenRouter API status
  • Try a different AI model
  • Enable Enhanced Detection in Privacy tab

Google Docs not working?

  • Switch to Aggressive injection mode
  • Enable Enhanced Detection
  • Refresh the Google Docs page
  • Open extension popup once before using
